# 001 - 番茄炒鸡蛋 (Tomato Scrambled Eggs)
**适合新人厨师的入门级下饭菜**

## 基本信息
- 难度：⭐️（非常简单，零失败）
- 制作时间：15 分钟
- 份量：2 人份

## 食材准备
| 食材 | 用量 | 备注 |
|------|------|------|
| 番茄 | 2-3 个（约 300g） | 选自然成熟的红番茄，味道更好 |
| 鸡蛋 | 3 个 | |
| 食用油 | 2 汤匙 | |
| 盐 | 约 3g（1小勺） | 可根据口味调整 |
| 白糖 | 约 3g（1小勺） | 中和番茄酸味，推荐添加 |
| 葱花 | 少许 | 点缀增香，可选 |

## 制作步骤
1. **处理食材**：番茄洗净后切成 2cm 左右的滚刀块；鸡蛋打入干净大碗，加入 1/4 小勺盐，用筷子充分搅打均匀。
2. **先炒鸡蛋**：平底锅烧热，倒入 1 汤匙食用油，油微微冒烟后倒入蛋液。等蛋液底部完全凝固后，用铲子轻轻从边缘往中间推，炒成大块鸡蛋，立刻盛出备用（鸡蛋不要炒太老，嫩一点更好吃）。
3. **熬出番茄汁**：不用刷锅，直接再加入 1 汤匙食用油，放入切好的番茄块，中火不停翻炒。炒到番茄变软后，用铲子轻轻按压番茄，帮助流出更多汁水。加入白糖，翻炒均匀后煮 1-2 分钟，让番茄的酸味和甜味融合。
4. **混合出锅**：把之前炒好的鸡蛋倒回锅里，让鸡蛋均匀裹上番茄汁，翻炒大约 30 秒就可以关火了。最后尝一下味道，不够咸再加一点盐，盛出后撒上葱花就完成啦！

## 厨师小技巧
- 这道菜最好吃的就是浓浓的番茄汁，非常下饭，多留一点汤汁拌米饭超绝。
- 如果喜欢吃口感更绵密的番茄，可以提前给番茄去皮：在番茄顶部划十字，用开水烫 1 分钟，皮就很好剥下来了。
- 新手不用担心失败，哪怕盐放多放少都好吃，是百分百不会翻车的入门菜！
